# 2026-04-18 实验室参数 / 演示参数双表 V1

## 参数双表
| 参数 | 实验室参数 | 演示参数 | 说明 |
|---|---:|---:|---|
| `capture_cooldown` | 1.5~2.5 | 2.5~4.0 | 演示场抑制重复抓拍 |
| `policy_capture_pending_window_s` | 6~8 | 8~12 | 演示时给锁定更多缓冲 |
| `LegalHoldMs` | 2000 | 2500~3000 | 演示场减少短时抖动误判 |
| `ReconfirmWindowMs` | 1200 | 1500~1800 | 演示场增强身份回稳 |
| 风险进入阈值 | 偏敏感 | 偏稳健 | 现场抗噪优先 |
| 视觉锁定阈值 | 偏快 | 偏稳 | 演示防抖优先 |

## 切换规则
1. 实验室参数用于调试和问题复现。
2. 演示参数用于现场展示与评审。
3. 每次切换后必须跑一轮 `preflight + acceptance`。
