# 2026-04-13 接线/供电/端口表 V1.1

## 1. 冻结口径（本周不再改）
1. 单节点拓扑：`Node A + 雷达 + 云台 + USB 摄像头(PC 侧)`。
2. 舵机与主控必须共地。
3. 舵机建议独立 5V 供电（与主控共地，不共电源回路）。
4. USB 摄像头固定接 PC，不直连 Node A。

## 2. 接线表（冻结）

| 模块 | 接到哪里 | 端口/引脚 | 备注 |
|---|---|---|---|
| ESP32-S3 Node A | PC | USB（当前口径 `COM4`） | 监控串口 `115200` |
| 雷达串口 RX/TX | Node A 串口1 | `RX=GPIO18`，`TX=GPIO17` | 雷达波特率 `256000` |
| 云台 Pan 舵机 | Node A PWM | `GPIO4` | 建议独立 5V 供电 |
| 云台 Tilt 舵机 | Node A PWM | `GPIO5` | 建议独立 5V 供电 |
| USB 摄像头 | PC USB | `source=1`（当前联调口径） | 由 `vision_bridge` 打开 |

## 3. 供电表（冻结）

| 模块 | 供电方式 | 要求 |
|---|---|---|
| Node A 主控 | USB 5V | 用于主控与串口联调 |
| 雷达 | Node A/外部稳定电源 | 与 Node A 共地 |
| Pan/Tilt 舵机 | 独立 5V（推荐 >=3A） | 与 Node A 共地；避免主控 USB 直接带舵机 |
| USB 摄像头 | PC USB | 与 Node A 电气隔离，数据在软件层汇聚 |

## 4. 端口与参数冻结

| 项目 | 冻结值 |
|---|---|
| Node A 串口号 | `COM4`（按当前现场口径） |
| 监控波特率 | `115200` |
| 雷达波特率 | `256000` |
| 网页服务端口 | `8765` |
| 基线版本号 | `Node_A_Base_Demo_V1.1` |

## 5. 启动顺序（防串口冲突）
1. 先启动 `node_a_serial_bridge`。
2. 再启动 `vision_bridge`。
3. 最后启动 `vision_web_server`。
4. 烧录前先停桥接，烧录后再恢复。
