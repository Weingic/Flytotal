# 2026-04-16 视觉参与主链规则 V1

## 1. 定位
视觉链不是独立主线，而是 Hunter 主线的增强模块。

## 2. 责任分工
1. 雷达：发现目标、提供粗跟踪与方位输入。
2. 云台：按雷达结果快速转向。
3. 视觉：近距锁定、稳定跟踪、抓拍增强。

## 3. 主链融合规则
1. 高风险/事件态优先触发抓拍。
2. 视觉成功锁定时，提高证据有效性（便于事件留档）。
3. 视觉未锁定不应阻断雷达主链；仅影响证据质量与显示语义。

## 4. 串口与网页联动
1. 串口状态输出：`vision_state/vision_locked/capture_ready`。
2. 网页同步显示并参与态势解释。

## 5. V1 不做项
1. 不训练复杂模型。
2. 不做通用无人机识别大模型路线。
3. 先保证闭环可运行，再谈精度优化。
