# 2026-04-16 雷达到云台映射规则 V1

## 1. 目标
把雷达坐标（x/y）转换为云台可执行的 `pan/tilt` 指令，实现“先粗对准”。

## 2. 输入与输出
1. 输入：雷达轨迹 `x_mm / y_mm / vx_mm_s / vy_mm_s`。
2. 输出：云台 `pan_deg / tilt_deg`（并做限幅与平滑）。

## 3. V1 规则
1. 以中心位（`CenterPanDeg/CenterTiltDeg`）为基准。
2. 按目标相对位置做比例映射（左偏->减 pan，右偏->加 pan）。
3. 速度项用于预测补偿（减少跟随滞后）。
4. 对输出角度执行安全限幅与步进平滑。

## 4. 验收口径
1. 雷达目标出现后，云台能转向目标区域（粗准即可）。
2. 不要求一次到位高精度，但不能明显反向或震荡失控。
