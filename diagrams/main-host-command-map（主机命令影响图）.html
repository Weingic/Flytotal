→

→

→

TRACK,CLEAR

告诉设备：“模拟目标结束了，清掉它。”

→

→
结果

设备后续就会回到无模拟目标或只看真实雷达的状态。

Status Commands
4. `STATUS` / `SELFTEST` / `SUMMARY` 是查什么的

STATUS

  • 查询“现在这一刻”的即时状态。
  • 比如主状态、风险、轨迹是否 active、事件 ID 等。

SELFTEST

  • 查询“设备配置和健康状态”。
  • 比如串口波特率、KP/KD、test mode、servo 状态等。

SUMMARY

  • 查询“从开始到现在的累计统计”。
  • 比如 track_active_count、event_opened_count、max_risk_score。

Reset Command
6. `RESET` 会影响什么
RESET

告诉设备“把很多运行时状态清掉”。

→
清理内容

Other Useful Commands
7. 其他你会见到的命令

控制与诊断类

  • SAFE：安全模式开关。
  • DEBUG：调试输出开关。
  • QUIET：减少输出。
  • UPLINK：上行输出开关。
  • DIAG,SERVO / DIAG,STOP：舵机诊断。

手动动作类

  • SERVO：手动设置舵机角度。
  • CENTER：回中。
  • PAN / TILT：单独调横向或纵向。
  • HANDOVER,...：处理交接请求或查看交接状态。
